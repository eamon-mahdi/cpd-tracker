# CPD Tracker

Continuing Professional Development Portfolio Tracker aligned with Royal College of Pathologists (RCPath) framework.

## Features
- Track CPD activities across 6 RCPath domains
- Multi-device sync via Chrome
- PDF and CSV export
- Professional portfolio reporting
- Responsive design (works on MacBook and iPhone)

## How to Use

1. Visit: https://cpd-tracker.vercel.app
2. Enter your professional details
3. Start adding CPD activities
4. Export as PDF or CSV

## Data & Backup

- Data stored locally on your device
- Monthly CSV backup recommended
- Chrome Sync for cross-device sync

## Created for Healthcare Professionals
Specifically designed for haematologists and pathologists using RCPath revalidation framework.
